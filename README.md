# Fitfamly — Legal Pages

Public legal pages for the **Fitfamly** Android app, hosted via GitHub Pages.

This repository is public **only** so that Google Play's crawler can reach these
pages without authentication. The Fitfamly application source code lives in a
separate private repository.

| Page | URL |
|---|---|
| Privacy Policy | https://venkaat92-rock.github.io/fitfamly-legal/ |
| Account & Data Deletion | https://venkaat92-rock.github.io/fitfamly-legal/data-deletion.html |

These URLs are referenced in the Google Play Console under
**App content → Data safety** (privacy policy URL and delete-data URL).
